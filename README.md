# Adachi Odontologia

Landing page institucional da Adachi Odontologia — clínica de odontologia sistêmica, biológica e humanizada em Santo André/SP, desde 2003.

## Stack

- HTML estático + CSS + JavaScript vanilla
- Tailwind CSS via CDN
- Plus Jakarta Sans (Google Fonts)
- Hospedagem: Vercel

## Estrutura

```
.
├── index.html              # página principal
├── styles.css              # estilos custom + animações
├── script.js               # interações (carrosséis, FAQ, form)
├── vercel.json             # config de cache/headers
└── assets/                 # imagens, logo, favicon
```

## Desenvolvimento local

```bash
python3 -m http.server 8000
```

Abra http://localhost:8000

## Deploy

Push para `main` → Vercel faz deploy automático.
